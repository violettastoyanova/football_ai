# Football Manager

## Описание
Конзолно приложение за управление на футболни клубове, играчи, трансфери, лиги и мачове. Използва SQLite база данни и чатбот с регулярни изрази.

## Технологии
Python 3, SQLite, Regex

## Валидации

### Клубове и играчи:
- Позиция: само Вратар, Защитник, Полузащитник, Нападател
- Номер: 1-99, уникален в клуба
- Дата на раждане: формат ДД-ММ-ГГГГ
- Не може да се изтрие клуб с играчи

### Трансфери:
- Играчът трябва да е в отбора, от който се трансферира
- Не може трансфер към същия клуб

### Лиги:
- Сезон: YYYY/YYYY (втората година е +1)
- Не може дублиране на лига с име и сезон
- За генериране на програма: минимум 4 отбора
- Не може премахване на отбор при генерирана програма

### Мачове:
- Минути за голове и картони: 1-120
- Тип картон: Y или R
- Максимум 1 червен картон на играч
- Голове и картони само след записан резултат

### Класиране:
- Само мачове с status = 'played' участват в изчисленията
- Мачове с NULL резултат се игнорират
- Отбори без изиграни мачове се показват с нули


## Алгоритми

### Round-Robin алгоритъм (за генериране на програма)
При четен брой отбори (N):
- Брой кръгове: N - 1
- Брой мачове: N * (N - 1) / 2

При нечетен брой отбори:
- Добавя се BYE (почивка)
- Брой кръгове: N

Отборите се разбъркват случайно при всяко генериране на програма.

### Класиране (Standings)
1. Вземат се всички отбори от league_teams
2. Вземат се всички изиграни мачове (status = 'played') от matches
3. За всеки мач се обновява статистиката на двата отбора:
   - MP (изиграни мачове)
   - W/D/L (победи/равни/загуби)
   - GF/GA (вкарани/допуснати голове)
   - PTS (точки)
4. Изчислява се GD (голова разлика) = GF - GA
5. Сортиране по:
   - Точки (низходящо)
   - Голова разлика (низходящо)
   - Вкарани голове (низходящо)
   - Име на отбора (възходящо)
6. Отличен критерий: При равни точки се прилагат директни срещи

- INFO - информативно съобщение

## Тестови сценарии за Класиране

1. Лига с отбори, без изиграни мачове - Таблица с нули, съобщение "Няма изиграни мачове"
2. Един изигран мач 2:1 - Правилни точки (3/0) и голове
3. Два мача - Коректно натрупване на статистика
4. Равни точки - Сортиране по GD/GF
5. Директни срещи при равни точки - H2H критерий (отличен)
6. Несъществуваща лига - Ясна грешка
7. Лига без отбори - Ясна грешка
